## 📌프로젝트 소개
<b>원하는 시간! 원하는 자리!</b> <br>
<b>주차 걱정 없이 편하게 주차하자!</b> <br>

<span style="background-color:#C0FFFF"> ParkIT</span>은 일상에서 주차로 인한 스트레스와 시간 낭비를 해결하고자 시작된 프로젝트 입니다. <br>
주차장 자리를 미리 예약하는 플랫폼으로 사용자들에게 주차 걱정 없는 효율적인 주차 경험을 제공합니다. <br>
<br>
